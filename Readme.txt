== Description ==

Плагин "Hide WordPress update notification" скрывает уведомления об обновлении WordPress в панели управления.

P.S: Если у вас остались какие-либо вопросы пишите на почту <cite>info[@]artemsannikov.ru</cite>.

== Installation ==
	
1. Загрузите плагин "Hide WordPress update notification" в директорию `/wp-content/plugins/`;
2. Активируйте "Hide WordPress update notification" на странице `Плагины` в панели управления.
